## DNS

域名系统（DNS）是Internet的电话簿。人类通过域名（如baidu.com或者bilibili.com）在线访问信息。Web浏览器通过Internet协议（IP）地址进行交互。DNS将域名转换为IP地址，因此浏览器可以加载Internet资源。
